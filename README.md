# Supply Chain Analysis Dashboard

## 📊 Project Overview
Analysis of supply chain operations for 
Skincare, Haircare & Cosmetics industry 
using Power BI and Excel.

## 🎯 Key KPIs
- Revenue Generated: 577.60K
- Total SKUs: 100
- Products Sold: 46K
- Avg Defect Rate: 2.28

## 🔍 Key Insights
- Skincare drives highest revenue
- Supplier 5 has highest defect rate
- Mumbai is top revenue location
- Carrier A & C have slowest shipping time
- Route C is most expensive shipping route
- Female customers generate more revenue
